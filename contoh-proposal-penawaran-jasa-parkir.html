1. sebutkan bagian-bagian dari proposal penawaran jasa desain!


Proposal dapat diartikan sebagai rencana kegiatan. Pada umumnya, proposal dapat dibagi menjadi 2 jenis yaitu proposal kegiatan dan proposal penawaran. Proposal kegiatan biasanya dimaksudkan untuk memberikan laporan tentang rencana diadakannya suatu kegiatan. Selain itu, proposal ini juga kerap digunakan untuk memohon bantuan dari pihak donatur, baik dalam bentuk dana maupun barang. Sementara itu, yang dimaksud dengan proposal penawaran adalah proposal yang menawarkan barang maupun jasa kepada pihak lain. Proposal jenis ini paling sering kita lihat dalam proses lelang.


Sebagai contoh, pemerintah ingin melakukan pengadaan kotak pencoblosan untuk keperluan Pemilihan Umum di TPS. Untuk memenuhi kebutuhan ini, pemerintah melakukan lelang. Kemudian, beberapa perusahaan mengirimkan proposal penawaran berisi barang yang dibutuhkan pemerintah beserta nominalnya. Keputusan akhirnya berada di tangan pemerintah untuk menentukan penawaran dengan harga terendah namun dengan kualitas terbaik.


Contoh lain dari proposal penawaran adalah proposal penawaran jasa desain

Berikut bagian-bagian dari proposal penawaran jasa desain yang harus dicantumkan:

1. Identitas pemberi penawaran

Informasi ini mencakup nama lengkap, alamat lengkap, kontak, informasi pendirian usaha (jika berbentuk badan usaha), alamat situs (jika ada) dll.


2. Jenis jasa desain yang ditawarkan (disesuaikan dengan permintaan klien), beserta spesifikasi jasa desain.


3. Biaya jasa desain yang ditawarkan


4. Waktu pengerjaan proyek


5. Portfolio pemberi penawaran

Hatta Afkar menemukan solusi yang saling menguntungkan bagi pemilik lahan parkir, pengguna parkiran dan tentunya bagi pajak negara melalui aplikasi bernama Parkiran.

 

“Semua mitra kami tertarik dengan program yang kami miliki. Salah satunya memberikan asuransi terhadap kendaraan yang parkir,” ujar pria 28 tahun ini. Dalam jumpa pers di Tanamera Coffee, Kebayoran Baru, Jakarta hari ini (27/1). Aplikasi ini menjadi solusi baru bagi para pengendara yang ingin memarkirkan kendaraannya dengan aman. “Kami pun mengedukasi pemilik lahan parkir untuk memberikan fasilitas tambahan seperti toilet, musholla, warung makan, wifi gratis, cucian motor dan helm agar memiliki nilai tambah bagi calon pengguna jasa layanan parkir,” tuturnya. Jadi pemilik lahan didorong menyediakan one stop service, selain membuat pelanggan senang, juga menambah penghasilan.

 

Ia meyakinkan, selain pengendara, para mitra yang menyediakan lahan parkir juga ikut kebagian manfaatnya. Karena dapat dipastikan mitra yang bergabung bersama aplikasi Parkiran mendapatkan nilai tambah dari lahan tidur yang dimiliki.

 

Saat ini sudah ada 25 mitra yang tersebar di Jabodetabek, 15 di antaranya sudah online. Dari 25 mitra tersebut, Parkiran telah mendapat jatah 25 persen dari rata-rata kapasitas 100 untuk motor dan rata-rata lima untuk mobil yang disediakan mitra. Ke depannya, Parkiran berkomitmen untuk memperluas mitra parkir dengan target 25 parkiran baru tiap bulannya.

 

“Untuk jatah parkir, saya ingin mengubah pola berpikir masyarakat yang tadinya konvensional bisa meningkatkan pendapatannya dengan pendekatan digital lewat aplikasi ini,” sambung Hatta.

Jika dalam 1 jam terisi penuh dan tak ada kendaraan yang keluar dan masuk . Pendapat maksimum tempat jasa parkir pak eka selama 1 jam adalah


kelas : X SMA
mapel : matematika
kategori : pertidaksamaan linier dua variabel
kata kunci : soal cerita

kode : 10.2.4 [matematika SMA kelas X bab 4 pertidaksamaan linier dua variabel]

Pembahasan:

untuk menyelesaikan soal seperti ini kita buat terlebih dahulu kalimat matematikanya, dan kita tentukan nilai untuk variabel x dan nilai untuk variabel y.
pertidaksamaan linier dua variabel adalah kalimat terbuka yang menggunakan 2 variabel, dan masing-masing variabel berderajat 1, serta menggunakan tanda penghubung <, >, ≤, ≥.
bentuk pertidaksamaan linier dapat ditulis:
ax + by ≤ c
ax + by ≥ c
ax + by < c
ax + by > c
dan hasilnya berbeda dibandingkan dengan persamaan, jika dalam persamaan hasilnya berupa himpunan pasangan titik-titik, namun jika pertidaksamaan linier dua variabel hasilnya berupa daerah arsiran

dalam soal, kita buat pertidaksamaannya terlebih dahulu:
mobil kecil = x
mobil besar = y
x + y ≤ 58 
6x + 24y ≤ 600
z = 3000x + 5000y
ditanya keuntungan maksimum

jawab:
kita buat nol fungsi terlebih dahulu
x + y ≤ 58
memotong sumbu x di titik y = 0
x + y = 58
x + 0 = 58
x = 58
memotong sumbu y di titik x = 0
x + y = 58
0 + y = 58
y = 58

6x + 24y ≤ 600 kita kecilkan dengan membagi 6 untuk semua ruas 
x + 4y ≤ 100
memotong sumbu x apabila y = 0
x + 4y = 100
x + 0 = 100
x = 100
memotong sumbu y apabila x = 0
x + 4y = 100
0 + 4y = 100
4y = 100
y = 100/4
y = 25

langkah selanjutnya kita eliminasi
x + 4y = 100
x + y = 58
--------------------- -
      3y = 42
           y = 42/3
           y = 14
x + y = 58
x + 14 = 58
x = 58-14
x = 44

kita tentukan titik uji yang memenuhi
kita dapat 3 titik uji, perhatikan gambar pada lampiran
1) (0,25)
2) (33,14)
3) (58,0)
subsitusikan titik uji pada nilai z = 3000x + 5000y
1) titik uji pertama (0, 25)
    z = 3000x + 5000y
      = 3000(0) + 5000(25)
      = 0 + 12500.000
      = 125.000
2) titik uji kedua (44, 14)
    z = 3000x + 5000y
       = 3000(44) + 5000(14)
       = 132.000 + 70.000
      = 202.000
3) titik uji ketiga (58, 0)
    z = 3000x + 5000y
      = 3000(58) + 5000(0)
      = 174.000 + 0
      = 174.000

pendapatan maksimum pak eka selama 1 jam Rp 202.000, pada saat yang parkir 44 mobil kecil dan 14 mobil besar

28. Di tempat parkir sebuah petokoan terdapat 105 kendaraan yang terdiri dari mobil dan sepeda motor. Jumlah roda seluruhnya ada 300 buah. Jika tukang parkir memperoleh uang jasa untuk tiap mobil dan sepeda motor berturut turut Rp5000 dan Rp2000 , maka pendapatan jasa tukang parkir saat itu adalah....


misalkan mobil = x motor = y
x + y = 105
x = 105 - y
4x + 2y = 300
4( 105 - y ) + 2y = 300
420 - 4y + 2y = 300
-2y = 300 - 420
-2y = -120
y = 60
x = 105 - 60 = 45
45(5000) + 60(2000) =
225.000 + 120.000 = 345.000
pendapatan tukang parkir saat itu =
Rp. 345.000,00
