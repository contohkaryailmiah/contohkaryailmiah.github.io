misal umur deksa = x
umur elisa = x - 4
umur firda = (x - 4) - 3 = x - 7
x + (x - 4) + (x - 7 ) = 58
3x - 11 = 58
3x = 58 + 11
3x = 69
x = 23
umur deksa = x = 23
elisa = x - 4 = 23 - 4 = 19
ttl umr mereka = 23 + 19 = 42 th
misalkan
deksa = d
elisa = e
firda = f

d = e+ 4
e = f + 3

d + e + f = 58
(e+4) + (f+3) + f = 58
2f + e = 51
2f + (f+3) = 51
f = 16

e = 16 + 3 = 19
d = 19 + 4 = 23

maka jumlah umur deksa dan elisa adalah 23+19 = 42 tahun

28. Umur Deksa 4 tahun lebih tua dari umur Elisa. Umur Elisa 3 tahun lebih tua dari umur Firda. Jika jumlah umur Deksa, Elisa, dan Firda adalah 58 tahun, jumlah umur Deksa dan Firda adalah ... .


x = umur Deksa

y = umur Elisa

z = umur Firda

maka : x = 4 + y.....persaman (1)

           y = 3 + z....persamaan (2)

           x + y + z = 58......persamaan (3)  

Jawab :

subsitusikan persamaan (1) dan (2) ke persamaan (3).

x + y + z = 58

(4 + y) + (3 +z) + z = 58

7 + y + 2z = 58

7 + (3 + z) + 2z = 58

10 + 3z = 58

3z = 58 - 10

3z = 48

z = 48/3

z = 16

* Kita subsitusikan nilai z = 16 ke persamaan...(2).

y = 3 + z

y = 3 + 16

y = 19

Subsituskan nilai y = 19 ke persamaan.....(1).

x = 4 + y

x = 4 + 19

x = 23

Jumlahkan nilai x dan z untuk mencari jumlah umur Deksa dan Firda.

x + z = 23 + 16

        = 39

Jadi, jumlah umur Deksa dan Firda adalah 39 tahun
