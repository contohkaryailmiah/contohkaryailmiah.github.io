18. contoh class student​


Jawaban: contoh class "Student" dalam bahasa pemrograman Python:

class Student:

def __init__(self, name, age, grade):

self.name = name

self.age = age

self.grade = grade

def study(self, subject):

print(f"{self.name} is studying {subject}.")

def take_exam(self, subject):

print(f"{self.name} is taking an exam in {subject}.")

def get_grade(self):

print(f"{self.name}'s grade is {self.grade}.")

# Contoh penggunaan class Student

student1 = Student("John", 17, "A")

student1.study("Math") # Output: John is studying Math.

student1.take_exam("Science") # Output: John is taking an exam in Science.

student1.get_grade() # Output: John's grade is A.
