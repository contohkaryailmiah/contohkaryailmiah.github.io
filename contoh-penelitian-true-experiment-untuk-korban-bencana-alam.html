Siklus pengeluaran merupakan rangkaian aktivitas bisnis yang melibatkan proses pembelian bahan baku atau inventaris dari pemasok luar untuk dijadikan barang jadi atau digunakan dalam operasional perusahaan. Siklus ini mencakup beberapa tahapan seperti permintaan pembelian, persetujuan anggaran/permintaan harga (request for quotation), pembuatan pesanan (purchase order), penerimaan barang/jasa (goods receipt), verifikasi faktur dan pembayaran kepada vendor.

Perbedaan utama antara survei/archival/laboratory experiment/field experiment adalah sebagai berikut:

- Survei: Metode ini dilakukan dengan mengumpulkan data dari partisipan menggunakan kuesioner atau wawancara untuk mendapatkan informasi subjektif.

- Archival: Metode ini menggunakan data sekunder yang sudah ada seperti catatan arsip, dokumen resmi, basis data publik untuk menjawab pertanyaan penelitian.

- Laboratory experiment: Metode ini dilakukan di lingkungan laboratorium dengan mengontrol variabel dan kondisi tertentu untuk mempelajari hubungan sebab-akibat antara variabel yang diteliti.

- Field experiment: Metode ini dilakukan di lapangan atau dalam konteks nyata, dimana peneliti melakukan intervensi atau perlakuan pada kelompok subjek untuk melihat dampaknya.

4. Buatlah fungsi untuk mendeteksi bilangan primer pada deret kemudian menghasilkan deret bilangan Boolean sesuai dengan input: Contoh: input = [17,18,12,11,10] Ouput:[true, false, false, true, false]


Jawaban:

def is_prime(n):

   """Fungsi untuk mendeteksi apakah bilangan n prima"""

   if n <= 1:

       return False

   for i in range(2, int(n**0.5) + 1):

       if n % i == 0:

           return False

   return True

def prime_boolean(arr):

   """Fungsi untuk menghasilkan deret bilangan Boolean sesuai dengan input"""

   boolean_arr = []

   for num in arr:

       boolean_arr.append(is_prime(num))

   return boolean_arr

# Contoh penggunaan

input_arr = [17, 18, 12, 11, 10]

output_arr = prime_boolean(input_arr)

print(output_arr) # Output: [True, False, False, True, False]

Fungsi is_prime digunakan untuk mendeteksi apakah suatu bilangan merupakan bilangan prima atau bukan. Fungsi ini menggunakan metode brute-force untuk memeriksa faktor-faktor bilangan tersebut.

Fungsi prime_boolean digunakan untuk menghasilkan deret bilangan Boolean sesuai dengan input. Fungsi ini memanggil fungsi is_prime untuk setiap bilangan dalam input, dan menghasilkan deret bilangan Boolean dengan nilai True jika bilangan tersebut prima, dan False jika tidak.

18. Hal apa saja yang perlu diperhatikan saat melakukan experiment


Jawaban:

Hal hal yang perlu diperhatikan dalam melakukan eksperimen diantaranya adalah memperhatikan Kesehatan dan keselamatan kerja praktikum, juga memperhatikan prosedur melakukan praktikum :

Kesehatan dan keselamatan kerja praktikum

a. Dalam eksperimen setiap siswa harus mengadakan percobaan dan bekerja sesuai prosedur/langkah kerja tertentu.

b. Agar eksperimen tidak gagal dan hasilnya tidak membahayakan, maka kondisi alat dan mutu bahan yang digunakan harus baik dan bersih.

c. Siswa perlu diberi petunjuk praktikum yang jelas sehingga dapat Menggunakan alat yang tepat sesuai dengan fungsinya, melakukan perawatan umum yang meliputi kebersihan dan keindahan tempat kerja.

d. Setiap pekerja/praktikan harus memahami situasi laboratorium dan diberikan pembelakan apa yang harus dilakukan dalam hal tindakan penyelamatan jika terjadi kecelakaan.

Prosedur eksperimen:

a. Perlu dijelaskan kepada siswa bagaimana tujuan eksperimen, dan apa yang harus dibuktikan melalui eksperimen.

b. Memberi penjelasan kepada siswa tentang alat-alat serta bahan-bahan yang akan dipergunakan dalam eksperimen, apa parameter yang perlu dikontrol maupun divariasikan, urutan eksperimen dan apa yang dicatat dalam hasil eksperimen.

c. Selama eksperimen berlangsung guru harus mengawasi pekerjaan siswa dan terbuka dengan saran dan pertanyaan dari siswa.

d. Setelah eksperimen selesai guru harus mengumpulkan hasil penelitian siswa, mendiskusikannya, dan mengevaluasi eksperimen tersebut

39. Will wedoan experiment


PART = TRANSLATION

-------------

Jawaban:

Will we do an experiment ? " Akankah kita mengerjakan sebuah percobaan (eksperimen) "

=> ini merupakan kalimat ajakan ataupun prediksi akan suatu hal yang akan dilakukan.

Penjelasan:

1. Will => "Akankah " merupakan modal auxiliary. Modal auxiliary ini fungsional. "Will" bisa digunakan untuk bentuk kalimat interrogative "future plan" dan bisa digunakan sebagai "polite request" ( permintaan sopan )

2. We => "kita/kami" sebagai subyek pada kalimat interrogative.

3. Do =>"mengerjakan/melakukan" sebagai verb ( kata kerja bentuk ke satu )

4. An => "sebuah / suatu" sebagai article ( imbuhan menyatakan quantiti )

5. Experiment => "percobaan/eksperimen" sebagai object ( noun / kata benda )
