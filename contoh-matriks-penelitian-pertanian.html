32. Buatlah program untuk menukarkan dua buah matriks 2x4 menggunakan method dan array dengan ketentuan: a. Terdiri dari 2 method: matriks1 dan matriks2 yang nilai kembaliannnya berupa array. b. Nilai matriks1: 3 1 4 2 5 3 Nilai matriks2: 9 4 6 8 7 1


Jawaban:

Berikut jawabannya dalam bahasa C:

#define _CRT_SECURE_NO_WARNINGS

#include<stdio.h>

void printArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

printf("Matrix 1:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix1[i][j]);

 }

 printf("\n");

}

printf("\nMatrix 2:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix2[i][j]);

 }

 printf("\n");

}

}

void swapArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

int temp;

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  temp = matrix1[i][j];

  matrix1[i][j] = matrix2[i][j];

  matrix2[i][j] = temp;

 }

 

}

}

int main()

{

int cols = 2;

int rows = 3;

int matrix1[3][2] =  

{

 {3,1},

 {4,2},

 {5,3}

};

int matrix2[3][2] =  

{

 {9,4},

 {6,8},

 {7,1}

};

printArray(cols,rows,matrix1,matrix2);

swapArray(cols,rows,matrix1,matrix2);

printf("\nAfter Swap\n\n");

printArray(cols,rows,matrix1,matrix2);

getchar();

return 0;

}

Penjelasan:

Program ini pertama menginisialisasi jumlah baris dan kolom. Kemudian matrix1 dan matrix2 diinisialisasi dengan nilai yang telah ditentukan. Untuk mengetahui perbedaan antara sebelum dan sesudah maka dibuatlah method printArray untuk mencetak matrix1 dan matrix2. Method printArray digunakan 2 kali untuk melihat nilai sebelum dan sesudah. Terakhir ada method untuk swap nilai matrix 1 dan matrix 2 dengan menggunakan bantuan variable temp untuk menyimpan nilai matrix 1 sebelum dioverwrite nilai matrix2 dan digunakan untuk mengoverwrite nilai matrix2.

SEARCA merupakan badan dari organisasi ASEAN yang dibentuk 27 November 1966 oleh anggota negara ASEAN yang berpusat di Los Banos, Filipina. Adapun tujuan dibentuk badan ini adalah sebagai upaya mendukung munculnya inovasi di bidang pertanian dan industri bahan pangan pedesaan di wilayah Asia Tenggara.

Adapun prioritas SEARCA di bidang pertanian dan industri bahan pangan seperti: 1) Model Agri-Bisnis untuk peningkatan produktivitas dan pendapatan, 2) Sistem pertanian berkelanjutan dan pengelolaan sumber daya alam, 3) Ketahanan pangan dan gizi, 4) Kepemimpinan dan transformasional untuk ISPA, 5) Gender dan keterlibatan remaja dalam ISPA, 6) Peningkatan ISPA menuju ketahanan iklim, 7) dan Penerapan EcoHealh/One Health untuk ISPA.
